# Библиотека для работы с ПК RastrWin3 (RUSTab)
Пакет RastrWin3 предназначен для работы с ПК RastrWin3 (RUSTab).

Для работы с RastrWin3 используются библиотека Python 3:
 - pywin32 (win32com)
 - OpenPyXl - для работы с Excel
